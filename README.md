## Instalasi

Pastikan Anda memiliki Node.js dan npm terinstal di sistem Anda sebelum melanjutkan.

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/nama-pengguna/nama-repositori.git
   
2. **Move to directory Project**
   ```bash
   cd nama-repositori

4. **Install aplication**
   ```bash
   npm install

6. **Run application**
   ```bash
   npm start
